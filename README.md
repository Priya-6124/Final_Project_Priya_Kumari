# Student Cyber Security Management System

## Objective

The purpose of this project is to manage student information and perform basic cyber security assessments using Python.

## Features

- Add Student
- View Students
- Search Student
- Delete Student
- Security Assessment
- Generate Security Report
- File Handling
- Password Strength Checker
- Username Generator
- Login Validation

## Technologies Used

- Python 3
- Text File Storage (students.txt)

## Program Flow

1. User selects an option from the menu.
2. Student records are stored in students.txt.
3. Security assessments generate scores.
4. Reports calculate averages and identify poor security practices.
5. Cyber security tools provide password checking, username generation, and login validation.

## Challenges Faced

- Managing file storage
- Updating records
- Implementing search functionality
- Designing security score calculation

## Learning Outcomes

- Python Programming
- Functions
- Lists
- Loops
- Conditional Statements
- File Handling
- Cyber Security Fundamentals
